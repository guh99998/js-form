# 7 Days of Code - JS e DOM

## Dia 1
Desenvolvimento do HTML de 1 título e 1 formulário (o formulário contém 1 campo para informar o nome da pessoa e outro campo para informar a data de nascimento), e também 1 botão para submeter o formulário. Vincular uma função no botão para pegar o valor informado nos campos nome e data de nascimento e imprimir eles no console.

---

## Dia 2
Aplicar um conjunto mínimo de validações para o formulário, como ele representa uma pessoa, com nome e data de nascimento, as regras de negócio serão:
- Um nome precisa ter no mínimo três letras.
- Um nome pode ter no máximo 120 letras.
- Um nome pode apenas conter letras, e não números.
- A data de nascimento precisa estar no formato DD/MM/AAAA, por exemplo: 30/01/2026.
- O mês informado deve estar entre 01 e 12.
