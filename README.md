# 7 Days of Code - JS e DOM

## Dia 1
Desenvolvimento do HTML de 1 título e 1 formulário (o formulário contém 1 campo para informar o nome da pessoa e outro campo para informar a data de nascimento), e também 1 botão para submeter o formulário. Vincular uma função no botão para pegar o valor informado nos campos nome e data de nascimento e imprimir eles no console.
